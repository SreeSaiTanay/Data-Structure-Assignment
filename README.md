# DSA ASSIGNMENT 2
This repository contains five C programs demonstrating stack and queue applications string reversal using a stack, where characters are pushed and then popped back in reverse order; balanced parentheses checking in an mathematical expression, where a stack ensures every opening bracket has a matching closing bracket; next greater element for each element in an array, implemented using an efficient stack‑based approach to find the first larger element to the right; printer queue simulation, which models a real‑world print spooler using a simple queue with operations to add documents, print the next document, and display pending documents; and a menu‑driven circular queue implemented using arrays, supporting enqueue, dequeue, peek, and display operations while reusing array space through circular index wrapping.
